# `underlying_or_strategy` 字段设计说明

这是一个非常关键、且直接影响后续产品建模与推荐效果的字段设计问题。

`underlying_or_strategy` 不是普通的产品说明字段，而是用于概括产品**底层资产与收益实现机制**的核心字段。它需要回答的，不只是“这个产品投什么”，更要回答“这个产品为什么会涨跌、主要靠什么赚钱、在组合中承担什么角色”。

在资产配置、产品推荐与组合管理场景下，这个字段会直接影响三类核心判断：

- **收益来源识别**：区分产品主要赚的是经济增长的钱、利率波动的钱、信用利差的钱，还是避险情绪与通胀预期的钱。  
- **风险暴露识别**：识别产品对利率、通胀、增长、汇率、信用风险、流动性环境等宏观变量的敏感性。  
- **组合角色识别**：判断该产品更适合作为核心配置、卫星增强、防御工具，还是流动性管理工具。  

因此，`underlying_or_strategy` 的设计不能停留在“产品名称解释”或“泛泛而谈的投资方向”层面，而应尽量满足以下要求：

- **结构化**：不同资产类别采用相对统一的表达框架，便于程序解析、规则筛选与后续标签化。  
- **可比较**：同类产品之间能够横向比较，例如同为债基时，可以直接比较久期、信用风险与收益驱动。  
- **具备经济含义**：字段内容应能够映射到真实的风险收益逻辑，而不是营销话术。  
- **服务配置决策**：研究、推荐、再平衡、风险监控等环节都能直接使用，而不是只能给人“读起来像简介”。  

换句话说，这个字段的目标不是“把产品描述得更完整”，而是“把产品的投资逻辑描述清楚、描述标准化、描述到可用于配置决策”。

---

## 一、权益类资产 (Equities / Stocks)

### ✅ 应包含内容：
| 内容 | 说明 |
|------|------|
| **跟踪指数名称** | 明确基准，如：沪深300指数、中证500指数、科创50指数等。 |
| **市值风格** | 如：“大盘股”、“中盘成长”、“小盘价值”，影响周期适配性。 |
| **行业分布特征** | 若有明显偏向，需注明，如：“偏重新能源与消费”、“科技权重超40%”。 |
| **是否为增强型策略** | 如：“被动指数跟踪+量化选股增强”或“纯全复制ETF”。 |

### 📌 推荐格式：
```text
[跟踪指数] + [市值风格] + （可选）[行业集中度或主题特征]
```

### 示例：
- `"跟踪沪深300指数，代表A股大盘蓝筹，金融与消费占比高"`
- `"跟踪中证500指数，聚焦中盘成长企业，行业分布较均衡"`
- `"跟踪科创50指数，聚焦半导体、AI与生物医药，高研发投入特征"`

> 💡 用途：判断其在复苏/过热阶段的表现潜力。

## 二、固定收益类资产 (Fixed Income / Bonds)

### ✅ 应包含内容：
| 内容 | 说明 |
|------|------|
| **债券类型** | 如：国债、政策性金融债（国开债）、信用债、城投债等。 |
| **久期范围或平均剩余期限** | 如：“平均久期约5-6年”、“短期限（<1年）”。 |
| **信用风险层级** | 如：“无信用风险（主权级）”、“AAA级为主”、“含少量AA+”。 |
| **收益驱动机制** | 如：“主要受益于利率下行”、“票息收入为主，资本利得有限”。 |

### 📌 推荐格式：
```text
[债券品种] + [久期水平] + [信用等级分布] + [收益模式]
```

### 示例：
- `"投资于10年期记账式国债，平均久期约6年，纯利率债，无信用风险"`
- `"主要持有国开行与农发行发行的政金债，平均剩余期限3-5年，信用等级AAA"`
- `"配置短期融资券与同业存单，剩余期限均小于1年，以获取稳定票息为主"`

> 💡 用途：区分利率风险暴露程度，决定其在“降息预期”下的配置价值。

## 三、现金及现金等价物 (Cash and Cash Equivalents)

### ✅ 应包含内容：
| 内容 | 说明 |
|------|------|
| **底层资产构成** | 如：银行存款、同业存单、逆回购、短融等。 |
| **流动性机制** | 如：“支持T+0快速赎回”、“T+1到账”。 |
| **安全性等级** | 如：“R1低风险，本金亏损概率极低”。 |
| **收益来源** | 如：“主要来自隔夜拆借与7天逆回购利率” |

### 📌 推荐格式：
```text
[底层资产] + [流动性安排] + [风险与收益特征]
```

### 示例：
- `"投资于高信用等级短期货币工具（如同业存单、逆回购），支持T+0实时赎回，R1风险等级"`
- `"配置AAA级短期融资券与银行存款，七日年化收益稳定，适合短期资金存放"`
- `"挂钩银行间市场7天质押式回购利率，浮动收益，无申购赎回费"`

> 💡 用途：明确其作为“流动性管理工具”而非“收益增强工具”的角色。

## 四、另类投资 (Alternative Investments)

> 在您提供的产品池中，典型代表为黄金ETF。

### ✅ 应包含内容：
| 内容 | 说明 |
|------|------|
| **价格锚定标的** | 如：“上海黄金交易所Au99.99现货合约”、“COMEX黄金期货主力合约”。 |
| **计价方式与汇率处理** | 如：“人民币计价，未对冲美元波动”、“以美元计价”。 |
| **实物支撑情况** | 如：“每份基金对应一定克数实物黄金”、“由商业银行托管”。 |
| **驱动因素说明** | 如：“受实际利率、通胀预期和地缘政治影响显著”。 |

### 📌 推荐格式：
```text
[锚定标的] + [计价货币] + [是否实物支撑] + [核心驱动逻辑]
```

### 示例：
- `"紧密跟踪上海黄金交易所Au99.99现货价格，人民币计价，基金资产由商业银行实物黄金托管"`
- `"挂钩COMEX黄金期货主力合约，美元计价，受美联储货币政策与美债实际利率主导"`
- `"非生息避险资产，与股票相关性低，在系统性风险上升时表现较强"`

> 💡 用途：帮助投资者理解其“非传统回报来源”的本质。

## 总结：`underlying_or_strategy` 字段的标准模板建议

| 资产类别 | 推荐信息结构 |
|--------|--------------|
| **权益类** | `跟踪[指数名称]，[市值风格]，[行业集中特征]` |
| **固收类** | `投资于[债券类型]，平均久期[数值]，信用等级[评级分布]，收益以[票息/资本利得]为主` |
| **现金类** | `配置[底层资产]，支持[T+0/T+1]申赎，R[1-2]风险，收益来自[利率类型]` |
| **另类投资** | `锚定[商品/衍生品]价格，[计价货币]，[实物/期货]支撑，受[驱动因素]影响大` |
