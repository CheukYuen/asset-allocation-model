# 🔥 **终极版兜底风险矩阵（推荐生产环境使用）**

## 📌 **数值设计逻辑（非常关键）**

* 保本产品风险最低，但仍需区分不同策略的底层资产性质
* 现金类最稳 → 固收类略有波动 → 另类结构性产品更高
* Sharpe 越高 → 表达风险极低、收益稳定
* max_drawdown_3y 用“轻微负值”，体现保本但仍有技术性波动
