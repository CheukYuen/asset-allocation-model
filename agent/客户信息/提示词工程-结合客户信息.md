# 🎯 最佳实践：四层嵌套式提示词架构（Four-Layer Prompt Framework）

我们可以将整个分析流程设计为 **四个层次的输入模块 + 一个综合指令**，形成一个“由外向内、由宏观到微观”的推理链条：

```
[Layer 1] 宏观数据输入 → 
[Layer 2] 美林时钟判断规则 → 
[Layer 3] 用户画像输入 → 
[Layer 4] 资产配置逻辑约束 → 
[Final Instruction] 综合生成报告
```
