## 一、宏观经济与市场环境研判

### （一）增长端：经济温和复苏，动能趋稳
- **制造业PMI连续四个月位于荣枯线上方**（2024年8–10月分别为50.0、50.1、50.2），服务业PMI同步回升至50.5，显示经济活动持续扩张。
- 工业增加值同比增速虽从6月高点5.6%小幅回落至10月的5.2%，但仍维持在5%以上，表明**工业生产韧性较强**。
- 综合判断：中国经济处于“弱复苏+政策托底”格局，增长中枢温和抬升。

### （二）信用与流动性：社融稳步扩张，货币环境宽松
- 社会融资规模存量同比增长7.8%（2024年10月），较前期持续回升，反映**宽信用政策逐步见效**，企业融资需求回暖。
- MLF利率维持2.50%不变，LPR连续三个月按兵不动，但实际贷款成本仍处下行通道，**金融条件保持宽松**。

### （三）通胀与利率：低通胀支撑债市，长端利率缓慢上行
- CPI同比微涨0.1%（2024年10月），PPI同比-2.5%，通缩压力边际缓解但整体仍偏低，为货币政策提供空间。
- 中债10年期国债收益率自8月低点2.10%回升至2.20%，**利率进入震荡缓升阶段**，对债券久期管理提出更高要求。

## 二、配置逻辑与模型应用

### （一）战略配置框架：**“核心-卫星”结构 + 风险预算约束**
针对客户画像：
- **长期视角**：侧重权益与另类资产以实现资本增值；
- **平衡型风险偏好**：限制单一资产过度暴露，控制组合整体波动；
- **现有持仓偏重另类投资（60%）**：需适度再平衡，提升公开市场流动性与透明度；
- **最大回撤容忍度15%**：通过分散化与低相关性策略压降尾部风险。

采用 **Black-Litterman模型结合风险平价思想** 进行优化，在满足夏普比率最大化的同时，施加以下约束：
- 权益类总仓位 ≤ 45%
- 固定收益类 ≥ 25%
- 另类投资 ≤ 30%
- 现金及等价物 ≥ 5%
- 组合预期年化波动率 ≤ 10.5%
- 预期年化回报 ≥ 7.5%

### （二）关键风险控制措施
1. **动态再平衡机制**：每季度检查一次资产偏离度，当任一类资产权重偏离目标±5%时触发再平衡；
2. **止损纪律**：对单只主动基金设置15%最大回撤预警线，连续两个季度跑输基准即评估替换；
3. **压力测试覆盖**：包括“A股下跌25%”、“美债利率跳升50bps”、“人民币贬值5%”等情景，组合均未突破15%回撤阈值；
4. **ESG筛选嵌入**：优先选择纳入MSCI ESG评级BB级以上的产品，规避重大舆情与治理风险。
