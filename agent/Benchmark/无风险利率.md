

下面我按**“大模型在智能投顾 Benchmark 中的真实使用方式”**来系统拆解，而不是只解释利率本身。

你给的这两条指标：

* **1 年期中债国债到期收益率**
* **10 年期中债国债到期收益率**

在传统投研里是“利率数据”，
但在**大模型 + 资产配置 Benchmark**体系中，它们会被**重构为一组“结构性决策信号”**，而不是被当成时间序列去拟合。

---

# 一、先给结论：大模型不会“预测利率”，而是用它们做 5 件事

在智能投顾的 Benchmark 体系中，这两条利率主要被大模型用于：

1. **无风险锚定（Risk-Free Anchor）**
2. **收益曲线形态识别（Term Structure Regime）**
3. **固收久期与风险预算的基准约束**
4. **股债相对性价比判断（ERP / Bond Attractiveness）**
5. **Benchmark 稳定性与解释性校准**

换句话说：
👉 **它们是“决策坐标系”，不是“预测目标”。**

---

# 二、第一层：作为 Benchmark 的「无风险与低风险锚」

## 1️⃣ 1 年期国债：无风险利率锚（Risk-Free Rate）

在智能投顾中，大模型会默认：

* **1Y 国债 ≈ 中国市场的近似无风险利率**
* 是所有“是否值得冒险”的比较基准

### 在 Benchmark 中的典型用法

| 用途                 | 大模型内部逻辑                         |
| ------------------ | ------------------------------- |
| 夏普 / 风险溢价判断        | 如果某资产长期回报 ≤ 1Y 国债 + 风险补偿 → 价值不足 |
| 现金 & 货币类 Benchmark | 现金类收益上限 ≈ 1Y 国债（扣费用）            |
| 防守配置下限             | 组合在极端防守状态，收益目标不应低于 1Y 国债        |

> **重要**：
> 大模型不会精确用 2.72% 还是 2.75%，
> 它关心的是：**“当前无风险回报是在 2.x 还是 4.x”**。

这是**量级判断，不是点估计**。

---

## 2️⃣ 10 年期国债：长期定价锚（Long-Term Discount Rate）

10Y 国债在 Benchmark 中承担的是：

* **长期贴现率**
* **固收类资产的“天花板锚”**

### 在 Benchmark 体系中的角色

| 资产   | 与 10Y 的关系               |
| ---- | ----------------------- |
| 纯债基金 | 长期回报不应显著高于 10Y 国债       |
| 固收+  | 超额收益需解释来源（信用 / 转债 / 权益） |
| 股票估值 | 股票的长期回报必须显著高于 10Y 才合理   |

---

# 三、第二层：收益率曲线 = 一个“宏观状态信号”

大模型真正关心的不是两个点，而是**它们的关系**。

## 关键派生指标：期限利差（Term Spread）

[
\text{Term Spread} = Y_{10Y} - Y_{1Y}
]

用你给的数据（2022-09-30）：

* 1Y = 2.43%
* 10Y = 2.68%
* **利差 ≈ +0.25%**

### 大模型会把它离散成「状态」

不是连续值，而是类似：

| 利差区间  | 大模型的语义标签    |
| ----- | ----------- |
| 明显正利差 | 正常增长 / 宽松预期 |
| 极窄    | 周期尾部 / 政策观望 |
| 倒挂    | 衰退 / 风险厌恶   |

在你的样本区间（2020–2022）：

* 利差 **持续收窄但未倒挂**
* 大模型会判断为：

> **“低增长、低通胀、政策偏宽松的中后周期环境”**

---

# 四、第三层：它如何影响四大类资产 Benchmark 权重

这里非常关键。

## 1️⃣ 对固收类 Benchmark 的影响

### 久期结构（而不是收益预测）

| 曲线状态 | Benchmark 偏好 |
| ---- | ------------ |
| 利差走扩 | 可容忍中长久期      |
| 利差收窄 | 偏中短久期        |
| 倒挂   | 极短久期 + 流动性   |

在你给的数据背景下：

* **1Y、10Y 同步下行**
* 利差缓慢收窄

👉 大模型会认为：

> 固收 Benchmark：
> **以中短久期为主，不激进拉久期**

---

## 2️⃣ 对股票 vs 债券 Benchmark 的影响（核心）

### 大模型内部其实在做一个判断：

> **“为了多赚 3–5%，是否值得承受 20% 的回撤？”**

它会隐式比较：

* 股票长期预期回报
* vs
* 10Y 国债的确定性收益

在你给的环境中：

* 10Y ≈ 2.7%
* 股票要求回报 ≥ 7–9%

如果宏观不支持高增长：

👉 **Benchmark 不会给股票极高权重**
👉 更偏“稳态配置”，而不是进攻态

# 六、在智能投顾 Benchmark 中的最终定位（一句话）

> **1 年期与 10 年期国债收益率，在大模型智能投顾中，不是“预测变量”，而是：**
>
> * 决定 **Benchmark 风险下限**
> * 决定 **固收久期与防守强度**
> * 决定 **股债性价比的判断坐标**
> * 决定 **组合是否处于“进攻 / 中性 / 防守”基准状态**
