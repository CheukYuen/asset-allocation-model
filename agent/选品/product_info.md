### ✅ 细颗粒度产品信息因子在资产配置中的决策权重（总和 = 100%）

|--------|---------|------|------|
| **一、风险-收益特征（共 45%）** | - | **45%** | 这是组合构建的核心输入，直接影响夏普比率和最大回撤控制。 |
| | 历史波动率（年化标准差） | 12% | 衡量风险的核心指标。低波动产品在防御阶段优先；高波动需匹配高预期回报。用于风险平价加权尤为关键。 |
| | 夏普比率（年化超额收益/波动率） | 10% | 综合衡量单位风险收益能力。在同类资产中优选高夏普产品（如两个债券基金选一个）。 |
| | 最大回撤（Max Drawdown） | 8% | 反映尾部风险承受能力，尤其在滞胀或衰退期至关重要。避免"幸存者偏差"下的虚假收益。 |
| | 历史收益率（年化） | 7% | 收益端基础输入，但不过度依赖——更关注风险调整后收益。长期复合增长能力体现管理质量。 |

| | 久期（Duration，针对债券） | 6% | 衡量利率敏感性。衰退期配长久期利率债；过热期配短久期或浮息债。 |
| | PPI弹性系数（历史回归估计） | 5% | 对工业金属、能源、原材料类资产尤为关键。PPI上行=企业盈利改善=上游受益。用于识别"真·周期品"。 |
| | GDP增长敏感性（Beta to PMI/CPI等） | 4% | 使用多因子回归估算产品对未来经济增长指标的响应程度。用于区分周期 vs 防御风格。 |

### 🔧 实战应用举例：

假设当前处于 **“复苏”阶段**（增长回升 + 通胀低位）：
1. **先按宏观敏感性筛选**：选出高贝塔（>1.1）、GDP弹性强的权益类产品（如中证500ETF、科技成长ETF）；
2. **再按风险收益优选**：在上述池中挑选夏普比率最高、波动率适中、最大回撤较小的产品；
3. **加入中国因素微调**：优先选择北向资金持续流入、且契合“新质生产力”政策主线的主题ETF；
4. **最后检查操作性**：剔除日均成交<3000万、费率过高或限购的产品；
5. **形成最终权重**：采用风险平价方式，结合波动率倒数加权，并设置单只上限。
