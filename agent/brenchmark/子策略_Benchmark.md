# **第一部分｜指标是否重复？答案：不重复，但必须正确拆分用途。**

# -------------------------------------

我们逐个分析：

---

# **1｜中位基准（Median Benchmark） → 不可删，是选品的“主心骨”**

### 作用：判断某产品是否“跑赢同类产品群体的整体表现”。

公式：

$$
\text{Median Benchmark} = \text{median}(r_1, r_2, \ldots, r_n)
$$

例如：固收+子策略

* 银行内部有 60 个固收+产品
* 过去 1 年收益中位数是 **4.8%**
* 某产品过去 1 年收益 **6%** → 跑赢中位基准

💡Summary
**中位基准是选品最核心的指标**，属于“绝对表现基准”。

---

# **2｜σ-based（μ ± σ 分布式基准） → 不可删，是产品评级分层工具**

### 作用：判断某产品在同类中处于哪个分层（Top、中位、落后）。

计算：

$$
\mu = mean(r_i), \quad \sigma = std(r_i)
$$

分档：

| 档位      | 定义                             | 含义   |
| ------- | ------------------------------ | ---- |
| Level 1 | ( r_p < \mu )                  | 同类偏弱 |
| Level 2 | ( \mu \le r_p < \mu + \sigma ) | 中等偏好 |
| Level 3 | ( r_p \ge \mu + \sigma )       | 同类强势 |

💡Summary
它的用途与中位数不同：
✔ 中位数用于“是否入围”
✔ σ-based 用于“入围后怎么排序 / 定级”

**两者不重复，功能互补。**

---

# **3｜时间截面指标（1Y / 2Y / 3Y） → 不可删，是“稳健性检测”**

### 作用：避免单一时间点误导，检查“表现是否持续”。

规则（极简）：

如果产品至少 **2 个时间段** 跑赢中位基准 → 合格
如果 **3 个都落后** → 淘汰
如果只在 **1 个** 时间段跑赢 → 边缘

三段时间分别起作用：

* **1Y**：最近表现（跟市场热点相关）
* **2Y**：更平滑（减少短期噪音）
* **3Y**：中长期管理能力

💡Summary
时间截面不是新的指标，只是给已有指标加一个“三期验证”。

# **第三部分｜最终可落地的指标体系（含使用场景）**

# -------------------------------------

我将所有指标按场景进行分类：

---

# **A｜场景 1：产品初筛（同类中是否合格？）**

### ★ 使用指标：**中位基准（Median Benchmark）**

规则：

* **1Y > median** → 合格
* **2Y > median** → 中期稳定
* **3Y > median** → 长期稳定

入围条件：

**至少 2 个时间段 > 中位基准**

---

# **B｜场景 2：产品定级（同类中属于什么水平？）**

### ★ 使用指标：**σ-based（μ ± σ 分布）**

等级划分：

| 水平          | 定义                             | 推荐用法     |
| ----------- | ------------------------------ | -------- |
| 强势（Top）     | ( r_p \ge \mu + \sigma )       | 作为备选推荐产品 |
| 中等（Neutral） | ( \mu \le r_p < \mu + \sigma ) | 作为可替代产品  |
| 弱势（Lagging） | ( r_p < \mu )                  | 剔除       |

**它不用于入围判断，而用于入围后的排序。**

---

# **C｜场景 3：组合构建阶段（映射子策略 → 产品池）**

### 用法：

* 中位基准：提供子策略最基本的“被动水平”
* σ-based：挑选“强势产品”作为卫星增强

例子：
固收+子策略，产品池 60 只：

* 中位基准 = 4.8%
* μ = 5.0%
* σ = 1.0%

某产品收益：

| 时间段 | 收益    | vs 中位 | vs μ+σ       |
| --- | ----- | ----- | ------------ |
| 1Y  | 6.2%  | ✔     | Level 3（Top） |
| 2Y  | 11.0% | ✔     | Level 2      |
| 3Y  | 15.5% | ✔     | Level 2      |

→ 入围（3/3 跑赢中位）
→ 排名靠前（含 1 次 Level 3）

# **第四部分｜所有指标的“使用案例示范（真实流程版）”**

# -------------------------------------

假设你筛选一个“固收+”基金 A。

---

## **Step 1：计算 1Y / 2Y / 3Y 同类中位数**

| 时间段 | 产品池收益分布（例子）          | 中位基准 |
| --- | -------------------- | ---- |
| 1Y  | [1%、2%、3%、...7%]     | 4.8% |
| 2Y  | [5%、6%、7%、...12%]    | 8.5% |
| 3Y  | [10%、12%、14%、...18%] | 15%  |

基金 A 的收益：

* 1Y = 6.2% → 跑赢
* 2Y = 11% → 跑赢
* 3Y = 15.5% → 跑赢

**Step 1 结果：入围**

---

## **Step 2：计算 μ、σ（只用于定级）**

假设：

* μ = 5.0%（1Y）
* σ = 1.0%

判定：

* 6.2% > (5.0% + 1.0%) → **Top（Level 3）**

**Step 2 结果：排序靠前**

---

## **Step 3：给投顾/用户的文案解释（最重要）**

> “该产品在过去 1/2/3 年均跑赢同类产品中位水平，并在最近一年处于同类收益的前 20% 区间（Top 档），适合作为该子策略的增强型选择。”

完全没有复杂指标，但意义清晰且直观。
