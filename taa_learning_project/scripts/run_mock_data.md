# 4. Functions to implement

Implement at least the following functions:

1. `get_paths() -> Tuple[Path, Path, Path, Path]`

```python
def get_paths() -> Tuple[Path, Path, Path, Path]:
    """
    Returns:
        script_path  : absolute Path of this script
        project_root : parent of the scripts/ directory
        data_dir     : project_root / "data"
        core_dir     : project_root / "core"
    """
```

2. `ensure_core_on_syspath(core_dir: Path) -> None`

* Adds `core_dir` to `sys.path` if not present.

3. `load_dataset_from_csv(data_dir: Path) -> Tuple[np.ndarray, pd.DataFrame, pd.Series]`

* Tries to load all three CSVs.

* Imports `create_mock_dataset` from `mock_data`.
* Calls it with `n_months=120`.

5. `load_or_create_dataset(data_dir: Path) -> Tuple[np.ndarray, pd.DataFrame, pd.Series, str]`

* Tries `load_dataset_from_csv`.
* If success:

  * returns the loaded objects and a mode string, e.g. `"loaded_from_csv"`.
* If failure:

  * calls `generate_and_save_dataset`, returns objects and `"generated_and_saved"`.

6. `print_summary(w_saa, returns_df, quadrants) -> None`

* Pretty-print:

  * SAA weights table
  * head() of returns
  * basic stats (optional)
  * quadrant value_counts()
